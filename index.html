🧠 WAIT, THIS APP IS ACTUALLY USEFUL NOW? 🧠

We accidentally made a mindful social media tool. It started as a joke, but people kept telling us the forced wait actually made them reconsider opening TikTok for the 47th time. So we added Mindful Mode — a suite of features disguised as ISP restrictions from 1997.

ISP Setting What It Actually Does
📞 Daily Connection Limits Set max connections per app per day. Hit the limit? "Your ISP has suspended service for today."
⏱ ISP Operating Hours Set a schedule for when you can connect. Outside hours? "The number you are calling is not in service."
🕑 Redial Cool-Down Force a wait between connections to the same app. 5 min to 1 hour. "Line is busy. Please wait."
📶 Line Quality Adjustable busy signal chance from 0% to 100%. Slide from "Crystal Clear" to "Terrible."
📈 Network Congestion The more you connect today, the slower the handshake gets. 10+ connections? Enjoy 2x dial-up speed.

All blocks happen BEFORE the dial-up starts. Instant feedback, maximum parody.

📖 CONNECTION LOG (EVENT VIEWER) 📖

Every connection attempt is recorded in the Event Viewer — a persistent, on-device log styled like the Windows 95 Event Viewer your IT guy used to stare at.

  • 📊 Summary stats: connections today, this week, all time, most-dialed apps
  • 🔍 Filter by outcome: Connected, Busy, Blocked, or Cancelled
  • 📄 Each entry shows the app emoji, name, timestamp, and outcome
  • 🔒 Stored entirely on your device — never transmitted anywhere

Finally, an answer to the question "How many times did I try to open Instagram today?" (The answer is always more than you think.)
